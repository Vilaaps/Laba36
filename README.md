## Описание проекта

**TaskBoard** — это интерактивная доска задач (аналог простого Trello), разработанная в рамках изучения полного цикла веб-разработки.

### Что умеет TaskBoard:

- Просматривать список задач 
- Добавлять задачи 
- Отмечать задачи как выполненные
- Редактировать задачи
- Удалять задачи
- Фильтровать задачи
- Смотреть счётчик выполненных задач
- Сохранять данные в базе SQLite

---

## Краткая инструкция 

**Запуск бэкенда:**

Откройте терминал в папке `TaskBoardApi` и выполните:
```bash
dotnet run
cd TaskBoardApi
dotnet restore
dotnet ef database update
```

**Запуск фронтенда**

Откройте папку `frontend` в VS Code, нажмите правой кнопкой на `index.html` и выберите "Open with Live Server".

## Главные выводы

1. **Fullstack-разработка** — это умение соединить фронтенд и бэкенд в одно работающее приложение.

2. **EF Core** сильно упрощает работу с базой данных — я пишу код на C#, а он сам превращает его в SQL.

3. **Асинхронность (async/await)** обязательна при работе с сетью и базой данных, иначе интерфейс будет зависать.

4. **Разделение кода на модули** (api.js для запросов и app.js для отрисовки) делает проект понятнее и проще в поддержке.

5. **Фильтрация на стороне клиента** работает быстрее, чем каждый раз ходить на сервер за новыми данными.
